# Goatify — Free Music Streaming
[![Build](https://github.com/chotathor/Goatify/actions/workflows/build.yml/badge.svg)](https://github.com/chotathor/Goatify/actions)
[![Stars](https://img.shields.io/github/stars/chotathor/Goatify)](https://github.com/chotathor/Goatify/stargazers)

> Your music, your way. Ad-free streaming powered by YouTube Music + Jio Saavn.

## Features
- YouTube Music & Jio Saavn streaming
- Spotify playlist import
- 320kbps audio quality
- Synced lyrics
- SponsorBlock
- Discord Rich Presence
- Offline playback
- Sleep timer

## Download
Get the latest APK from [Releases](https://github.com/chotathor/Goatify/releases)

## Tech Stack
- Kotlin & Jetpack Compose
- Material You design
- Gradle build system
